# 订单Excel批量处理工具
纯前端Excel处理网页，适配11项订单处理规则，浏览器本地运算，数据不上传。

## 使用步骤
1. 下载3个文件：固定A模板、业务B源文件、C公式文件
2. 网页分别上传三个Excel，填写店铺URL
3. 点击【开始批量处理】，自动下载修改后的A文件

## 文件规则说明
1. A：结果模板，前两行绝对不会被程序修改，保护表头格式
2. B：原始业务数据，程序自动清洗、筛选、改日期、替换字符
3. C：计算公式文件，A1单元格填写计算表达式；使用{{val}}指代B订单ID值
   示例：`{{val}}*2+10`

## 修改配置方法
打开index.html，顶部CONFIG大括号内：
- 修改列字母、关键词BRL/TK/已送达
- 修改锁定行数、日期格式、替换字符
无需改其他代码

## GitHub部署
1. 仓库上传 index.html + README.md
2. 仓库设置 → Pages → Build source选Deploy from a branch
3. 分支main，文件夹/(root)，保存；等待1分钟即可线上访问
